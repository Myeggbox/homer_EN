Foire Aux Questions

Comment fonctionne le bilan proposé par Homer?

Homer analyse les informations délivrées par votre assureur ainsi que le patrimoine que vous avez enregistré dans l'application Myeggbox, et couple ces informations afin de calculer pour vous si votre contrat d'assurance est optimisé selon vos besoins. Vos garanties, couvertures et exclusions sont affichées le plus simplement possible sous forme d'un tableau de bord expliquant tout ce que vous avez à savoir.

Mon assureur est-il compatible avec Homer?

Homer est compatible avec tous les assureurs Français. La synchronisation automatique est disponbile pour un grand nombre d'assureurs. Il est également possible d'ajouter manuellement vos conditions particulières et générales afin de permettre à Homer de réaliser son analyse. Pour le moment, la synchronisation automatique n'est pas disponible pour les assurances bancaires.

Quelle est la différence entre l’offre gratuite et l’offre payante?

Vous pouvez gratuitement accéder aux factures, notices, accessoires et consommables de vos biens les plus importants. Pour 3,29€/mois, Homer (l'assistant personnel présent dans l'application) vous accompagne dans l'optimisation de la protection de ces biens, en plus de vous renseigner sur les différents types de couverture disponibles.

Homer est-il disponible sur mon ordinateur?

Pour le moment, Homer est exclusivement disponible sur l'application mobile Myeggbox. Votre espace web Myeggbox vous permet d'accéder à vos documents importants.
